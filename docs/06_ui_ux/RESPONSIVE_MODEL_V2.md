## 1. Active Rule

Responsive behavior must remain model-backed and reproducible by preview and export. Existing detailed content is archived in `docs/09_archive/deprecated/RESPONSIVE_MODEL_V2.md`.

## 2. Contract

1. Preview and export must use the same scene dimensions and responsive settings.
2. Responsive behavior must not depend on untracked WebView defaults.
3. Validation must report unsupported responsive states before export.
